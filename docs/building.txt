Directory Structure:
    assets: Raw Assets.
    bin/<platform>: (Empty) Built binaries and supporting libraries/etc.
    deps: (Empty) Compile-Time Dependencies.
    dev/<platform>: Dev helpers/scripts for the platform.
    docs: Documentation.
    release: Everything needed to run the game, minus the binaries.
    src: Source Code.

Compile-Time Dependencies:
    GLAD: OpenGL Loader/Generator web service (see GLAD Options below).
    LLVM v10.0.0: C/C++ compiler.
    SDL2 v2.0.10: Platform/OpenGL functions.
    
    Win64:
        Mingw-w64 v8.1.0-x86_64-posix-seh-rev0: 

GLAD Options:
    OpenGL loader generated by glad 0.1.33 on Thu Jan  2 08:10:35 2020.

    Language/Generator: C/C++
    Specification: gl
    APIs: gl=3.3
    Profile: core
    Extensions:
        
    Loader: True
    Local files: False
    Omit khrplatform: False
    Reproducible: False

    Commandline:
        --profile="core" --api="gl=3.3" --generator="c" --spec="gl" --extensions=""
    Online:
        https://glad.dav1d.de/#profile=core&language=c&specification=gl&loader=on&api=gl%3D3.3

1) Install Compile-Time Dependencies to deps.
2) Put all libs/includes/srcs into deps/lib, deps/include, deps/src.
3) Run build script in dev/<platform>.
